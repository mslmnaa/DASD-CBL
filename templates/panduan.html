Apa fungsi aplikasi ini?

Sistem analitik berbasis machine learning untuk memperkirakan arah & level nilai tukar USD/IDR hari kerja berikutnya, sebagai pendukung keputusan treasury (kapan melakukan lindung nilai/konversi valuta). Model memprediksi return harian, lalu level direkonstruksi dengan USDIDR_besok = USDIDR_terakhir × exp(return).

Fokus pada arah (naik/turun) — itu sinyal paling andal (~65% benar). Angka level adalah estimasi, bukan kepastian.

Cara memakai fitur Prediksi & Simulasi

Langkah

  • 1. Buka halaman Prediksi. Bagian A menampilkan prediksi otomatis dari data terbaru — tanpa input apa pun.
  • 2. Untuk simulasi "bagaimana jika", isi nilai indikator makro di bagian B. Kotak sudah terisi otomatis dengan nilai terakhir.
  • 3. Ubah hanya indikator yang ingin Anda uji, biarkan sisanya (= dianggap tetap).
  • 4. Klik Hitung prediksi skenario → hasil muncul di kanan.
  • 5. Klik Reset untuk kembali ke nilai terakhir.

Rentang input yang wajar

    {% for col, rng in batas.items() %}
  • {{ col }} ({{ satuan[col] }}) — {{ "%g"|format(rng[0]) }}…{{ "%g"|format(rng[1]) }} · terakhir {{ "%g"|format(last[col]) }}
    • {% endfor %}
Input negatif, nol, di luar rentang, bukan angka, atau lompatan >25% dalam sehari akan ditolak demi keamanan & stabilitas aplikasi.

Cara membaca hasil

  • ▲ NAIK — Rupiah diprediksi melemah (butuh lebih banyak Rupiah per 1 USD).
  • ▼ TURUN — Rupiah diprediksi menguat.
  • Selisih = perkiraan perubahan nominal vs harga terakhir.
  • Confidence band (di Forecast) melebar mengikuti horizon — makin jauh, makin tidak pasti.

Catatan keamanan & etika

  • Tidak ada kredensial di kode (SECRET_KEY via environment variable).
  • Validasi input mencegah error & manipulasi data.
  • Disclaimer: alat bantu perencanaan, bukan saran finansial / kepastian.
